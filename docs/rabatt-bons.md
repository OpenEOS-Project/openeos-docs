# Rabatt-Bons

**Rabatt-Bons** sind Gutscheine oder Vergünstigungen, die das Kassenpersonal beim Bezahlen auswählen kann – etwa ein Helfer-Rabatt oder ein Freigetränk-Bon. Sie konfigurieren die verfügbaren Bons zentral, damit sie an allen Kassen einheitlich zur Verfügung stehen.

Sie finden den Bereich über **Rabatt-Bons** in der Seitenleiste.

![Rabatt-Bons](/img/screens/de/discounts.png)

## Rabatt-Bon erstellen

1. Klicken Sie auf **Rabatt-Bon erstellen**.
2. Legen Sie den Bon mit Bezeichnung und Rabattwert an.
3. Speichern Sie den Bon – er steht anschließend an der Kasse zur Auswahl.

![Rabatt-Bon erstellen](/img/screens/de/discounts-dialog.png)

## Einlösen an der Kasse

Beim Abschluss einer Bestellung wählt das Kassenpersonal den passenden Rabatt-Bon aus. Der Rabatt wird auf die Bestellung angerechnet und erscheint später in der [Auswertung](./auswertung.md).

:::tip[Typische Anwendungsfälle]
Rabatt-Bons eignen sich z. B. für Helferbons, Sponsoren-Gutscheine oder Sonderaktionen während des Festes.
:::
