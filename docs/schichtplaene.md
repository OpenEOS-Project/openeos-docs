# Schichtpläne

Mit **Schichtplänen** organisieren Sie die Helferinnen und Helfer Ihrer Veranstaltung. Sie erstellen Schichten und teilen Ihr Team ein – so weiß jeder, wann und wo er eingesetzt ist.

![Schichtpläne](/img/screens/de/shifts.png)

## Schichtplan erstellen

1. Klicken Sie auf **Neuer Schichtplan** bzw. **Ersten Schichtplan erstellen**.
2. Legen Sie die Schichten mit Zeitfenstern und Aufgaben/Bereichen an.
3. Teilen Sie Helfer den einzelnen Schichten zu.
4. Speichern Sie den Schichtplan.

![Schichtplan erstellen](/img/screens/de/shifts-dialog.png)

## Einsatz im Team

Schichtpläne schaffen Transparenz darüber, wer wann an welcher Station arbeitet. In Kombination mit den [Berechtigungen](./mitglieder.md) können Sie z. B. einer Schichtleitung gezielt nur das Modul **Schichtpläne** freigeben.

:::tip[Frühzeitig planen]
Erstellen Sie Schichtpläne rechtzeitig vor dem Fest, damit alle Helfer ihre Einsatzzeiten kennen und Sie Lücken früh erkennen.
:::
