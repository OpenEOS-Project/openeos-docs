# Mitglieder & Berechtigungen

Unter **Mitglieder** verwalten Sie das Team Ihrer Organisation. Sie laden neue Mitglieder ein und legen fest, auf welche Bereiche jedes Mitglied zugreifen darf.

![Mitgliederübersicht](/img/screens/de/members.png)

## Rollen

OpenEOS kennt zwei Rollen:

- **Administrator** – hat automatisch Zugriff auf **alle Module** und kann die Organisation vollständig verwalten.
- **Mitglied** – erhält gezielt einzelne Module freigeschaltet (siehe Berechtigungen).

In der Übersicht sehen Sie pro Person Name, E-Mail, **Rolle**, freigeschaltete **Berechtigungen** und das Beitrittsdatum.

## Mitglied einladen

1. Klicken Sie auf **Mitglied einladen**.
2. Geben Sie die **E-Mail-Adresse** der Person ein.
3. Entscheiden Sie über den Schalter **Administrator**, ob die Person Vollzugriff erhält.
4. Ist die Person kein Administrator, wählen Sie über die **Berechtigungen** die freizugebenden Module aus.
5. Klicken Sie auf **Mitglied einladen**.

![Mitglied einladen](/img/screens/de/members-dialog.png)

## Modulbasierte Berechtigungen

Für Mitglieder lassen sich die folgenden Module einzeln freischalten:

- **Produkte**
- **Events**
- **Geräte**
- **Mitglieder**
- **Schichtpläne**
- **Rabatt-Bons**
- **Pfand**
- **Auswertung**
- **Inventur**

Nur freigeschaltete Module erscheinen in der Navigation des jeweiligen Mitglieds. So geben Sie z. B. einer Schichtleitung nur die **Schichtpläne** frei, ohne Zugriff auf Auswertungen oder Geräte.

:::tip[Prinzip der minimalen Rechte]
Vergeben Sie nur die Berechtigungen, die jemand wirklich benötigt. Den Administrator-Vollzugriff sollten nur wenige, verantwortliche Personen erhalten.
:::
