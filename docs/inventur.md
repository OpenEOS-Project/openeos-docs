# Inventur

Mit der **Inventur** erfassen Sie Ihre Bestände und behalten die Lagerkontrolle für eine Veranstaltung im Blick. So wissen Sie jederzeit, wie viel Ware vorhanden ist und wann nachbestellt werden muss.

:::warning[Aktive Veranstaltung nötig]
Auch die Inventur bezieht sich auf die aktive Veranstaltung. Aktivieren Sie bei Bedarf zuerst eine [Veranstaltung](./veranstaltungen.md).
:::

![Inventur](/img/screens/de/inventory.png)

## Inventur anlegen

1. Klicken Sie auf **Neue Inventur**.
2. Erfassen Sie die Anfangsbestände Ihrer Produkte.
3. Speichern Sie die Inventur, um den Bestand festzuhalten.

## Bestände im Betrieb

Während des Verkaufs verringert sich der erfasste Bestand entsprechend der Verkäufe. So erkennen Sie frühzeitig, welche Produkte knapp werden. Der aktuelle Bestand wird auch in der [Produktliste](./produkte.md) in der Spalte **Bestand** angezeigt.

:::tip
Führen Sie vor dem Fest eine Anfangsinventur durch und nach dem Fest eine Schlussinventur. Die Differenz hilft Ihnen, Verbrauch und Schwund nachzuvollziehen.
:::
