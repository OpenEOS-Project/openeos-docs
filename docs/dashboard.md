# Dashboard

Das **Dashboard** ist Ihre Startseite nach der Anmeldung. Es gibt Ihnen einen schnellen Überblick über die aktuelle Geschäftstätigkeit Ihrer Organisation.

![Dashboard](/img/screens/de/dashboard.png)

## Kennzahlen

In den Kacheln am oberen Rand sehen Sie auf einen Blick:

| Kachel | Bedeutung |
| --- | --- |
| **Bestellungen heute** | Anzahl der heute erfassten Bestellungen (im Vergleich zu gestern) |
| **Umsatz heute** | Heutiger Umsatz (im Vergleich zu gestern) |
| **Aktive Events** | Anzahl der aktuell laufenden Veranstaltungen |
| **Aktive Benutzer** | Aktuell online angemeldete Benutzer |

## Letzte Aktivitäten

Darunter listet der Bereich **Letzte Aktivitäten** Ihre neuesten Bestellungen und Transaktionen auf. Solange noch keine Bestellungen erfasst wurden, erscheint hier ein Hinweis – starten Sie eine Veranstaltung, um Bestellungen anzunehmen.

## Dashboard anpassen

Über die Schaltfläche **Anpassen** oben rechts können Sie die angezeigten Inhalte des Dashboards an Ihre Bedürfnisse anpassen.

![Dashboard anpassen](/img/screens/de/dashboard-customize.png)

## Aktive Veranstaltung

Links in der Seitenleiste sehen Sie die aktuell **aktive Veranstaltung**. Viele Bereiche – etwa Produkte, Standorte und Inventur – beziehen sich immer auf das gerade aktive Event. Steht dort „Kein aktives Event", aktivieren Sie zunächst eine Veranstaltung (siehe [Veranstaltungen](./veranstaltungen.md)).
